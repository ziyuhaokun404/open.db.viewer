# Open DB Viewer MVP Smoke Checklist

## 1. 连接管理

- [ ] 首页可以加载已保存连接列表
- [ ] 点击“新建连接”可以进入连接表单页
- [ ] SQLite 连接输入名称和数据库文件后可以测试连接
- [ ] SQLite 连接保存后会回到首页并显示成功提示

## 2. SQLite 主链路

- [ ] 打开 SQLite 连接后可以进入数据库浏览页
- [ ] 对象树能够正常显示数据库与表节点
- [ ] 选择表节点后可以看到表结构
- [ ] 选择表节点后可以看到表数据分页区域
- [ ] 执行默认 SQL 可以看到查询结果和耗时
- [ ] 表数据支持导出 CSV
- [ ] 查询结果支持导出 CSV

## 3. 状态反馈

- [ ] 连接列表加载时会出现加载态
- [ ] 连接测试成功时会出现成功提示
- [ ] 连接测试失败时会出现错误提示
- [ ] 未选择表时会出现空态提示
- [ ] 无查询结果时会出现空态提示

## 4. 多数据库预留

- [ ] MySQL 连接表单字段显示完整
- [ ] PostgreSQL 连接表单字段显示完整
- [ ] MySQL / PostgreSQL 连接测试会返回当前占位实现的结果
